# AgroConnect Backend API

AgroConnect is a digital marketplace connecting farmers with buyers.

## Built With

- Python 3.13
- Django 6
- Django REST Framework
- JWT Authentication

## Features

- User Registration & Login
- Crop Marketplace
- Crop Image Upload
- Cart System
- Order System
- Farmer Dashboard
- Search & Pagination
- API Documentation

## API Documentation

Visit:

/swagger/

## Example Endpoints

POST /api/register/  
POST /api/login/  
GET /api/crops/  
POST /api/cart/  
POST /api/orders/create/
